# Barrier Dunes

Modern, beachy, simple, and classy website for the Barrier Dunes community at Cape San Blas.

## Tech Stack
- **Astro**
- **Tailwind CSS**
- **Lucide Astro** (Icons)

## Branding
- **Serif Font:** Playfair Display
- **Sans-Serif Font:** Montserrat
- **Colors:**
  - Sky Blue: `#D8EBF5`
  - Seafoam: `#9AD9D9`
  - Sandy Beige: `#D6C7A1`
  - Slate Gray: `#455A64`

## Project Structure
- `src/layouts/Layout.astro` - Global layout and Google Fonts
- `src/components/Navigation.astro` - Header and desktop/mobile nav
- `src/components/Footer.astro` - Footer with contact info and links
- `src/pages/index.astro` - Home page with hero and highlights
- `src/pages/rules-and-regulations.astro` - Detailed community guidelines
- `src/pages/local-attractions.astro` - Explore Gulf County
- `src/pages/local-resources.astro` - Safety and directories
- `src/pages/lost-and-found.astro` - Community reporting
- `src/pages/barrier-dunes-resources.astro` - Renter info and Clubhouse rental

## Getting Started
```bash
npm install
npm run dev
```

## Building
```bash
npm run build
```
